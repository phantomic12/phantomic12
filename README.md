## Yoav — Systems & Automation Engineer

NYC-based. I build tools that solve real problems — Linux distros, Windows deployment pipelines, network clients. Self-taught, practical, allergic to over-engineering.

### What I've built

**[WinForge](https://github.com/win-forge/winforge)** — Reusable GitHub Actions pipeline that builds custom Windows ISOs from UUP-dump. Injects Intel RST drivers, bypasses Win11 TPM/CPU checks, auto-uploads to gofile. Two-repo architecture (tool + user configs). MIT.

**[pdanet-linux](https://github.com/phantomic12/pdanet-linux)** — Linux client for PDA Net WiFi Direct Hotspot. CLI + libadwaita GUI, multi-distro packaging (Arch/Debian), polkit integration, SSID-aware auto-toggle. MIT.

**[phantomsarch](https://github.com/phantomic12/phantomsarch)** — Arch Linux spinoff with custom ISOs and package repository. Multi-edition builds, SourceForge distribution.

**[phn-repo](https://github.com/phantomic12/phn-repo)** — Automated AUR package builder via GitHub Actions. 10+ parallel workflows building AUR packages, split across repos to work around GA runner disk limits.
